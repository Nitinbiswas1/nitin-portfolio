# Customer Churn Prediction
## Mar 2025

Developed a machine learning model to predict telecom customer churn using usage patterns, contract details, and demographic data from a dataset of 1,00,000+ customers.

Conducted EDA by handling missing values, detecting outliers, and analyzing feature correlations, improving data quality and identifying key churn factors such as tenure and contract type.

Trained and evaluated models using ROC-AUC, Precision, Recall, and F1-score, achieving around 0.80+ predictive performance.
